# Civic AI

**Form Discovery & Preparation Assistant**

An AI-powered civic services assistant that helps users find the right government form, shows required document checklists, and generates downloadable blank PDF forms.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Tambo AI (Generative UI SDK)

## Getting Started

```sh
# Install dependencies
npm install

# Set up your Tambo API key in .env
# VITE_TAMBO_API_KEY=your_key_here

# Start the development server
npm run dev
```

## Features

- Natural language service discovery powered by Tambo AI
- Document checklist with progress tracking
- PDF form generation and download
- Generative UI components rendered by AI
